<h1>基本数据类型和引用类型</h1>

```
/*
 *基本数据类型
 *    String  Number  Boolean  Null Undefined
 *
 *引用数据类型
 *Object
 *
 *
 * JS中的变量都是保存到栈内存的。
 *      基本数据类型的值直接在栈内存中存储
 *      值与值之间是独立存在，修改一个变量不会影响其他的变量
 *
 *
 *对象是保存到堆内存中的，每创建一个新的对象，就会在堆内存中开辟出一个新的空间。而变量保存的是对象的内存地址（对象的引用）。
 */   

 var a=123;
 var b=a;
 a++;

 <!-- console.log("a=" +a);
 console.log("b=" +b); -->

 var obj=new Object();
 obj.name="孙悟空";

 var obj2=obj;

 //修改obj的name属性
 obj.name="猪八戒";

 console.log(obj.name);
 console.log(obj2.name);
 ```
